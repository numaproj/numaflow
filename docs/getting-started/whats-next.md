# What's Next

You have deployed both a [MonoVertex](monovertex.md) and a [Pipeline](pipeline.md). Here is where to go next.

## Try More Examples

Explore the full set of manifests in the [`examples`](https://github.com/numaproj/numaflow/tree/main/examples) directory, including reduce, sessions, joins, fallback sinks, and more.

## Explore Sources and Sinks

Numaflow ships with several built-in sources and sinks. A few to start with: the [Kafka](../user-guide/sources/kafka.md) source, the [HTTP](../user-guide/sources/http.md) source, the [Kafka](../user-guide/sinks/kafka.md) sink, and the [Log](../user-guide/sinks/log.md) sink. See the [Sources overview](../user-guide/sources/overview.md) for the complete list.

More importantly, when a built-in source does not fit your system, you can write your own: see [User-Defined Sources](../user-guide/sources/user-defined-sources.md).

## Write Your Own Logic

Learn how to write [User-defined Functions](../user-guide/user-defined-functions/user-defined-functions.md), transformers, sources, and sinks using the [Numaflow SDKs](../user-guide/sdks/overview.md).
